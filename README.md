# Chrome Extension - Persona Switcher

This Chrome extension allows you to select a persona and apply CSS changes to the current webpage, giving it a different "personality".

## Features

-   Select from a set of pre-defined personas.
-   Each persona applies unique CSS changes to the page.
-   Easily reset the page to its original styling.

## Personas

The extension will come with a few pre-defined personas, such as:

-   Lowercase Monospace: Applies `text-transform: lowercase` and `font-family: monospace`.
-   Uppercase: Applies `text-transform: uppercase`.
-   [Add more personas here]

## How to use

1.  Install the extension from the Chrome Web Store (once available).
2.  Open the extension popup.
3.  Select a persona from the dropdown menu.
4.  The CSS changes will be applied to the current page.
5.  To reset the changes, select the "Reset" option.
